設定手順
  1. Google Cloud Console でプロジェクトを作成（または選択）
  2. 「APIとサービス」→「ライブラリ」→「Google Drive API」を有効化
  3. 「認証情報」→「認証情報を作成」→「APIキー」をクリック → コピーして上に貼り付け
  4. （推奨）APIキーの制限で「Drive API」のみ許可
  5. Google Drive で音源フォルダを右クリック →「共有」→「リンクを知っている全員」に設定
  6. フォルダのURLをコピーして上に貼り付け → 「接続してファイルを取得」
⚠ Google Drive API は HTTPS（またはlocalhost）が必要です。 ローカルでは python -m http.server 8000 で起動してください。
